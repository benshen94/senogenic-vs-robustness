# Saved Results Index

This folder records generated artifacts for the threshold/noise aging project.

Use `outputs.csv` for durable entries. The expected columns are:

- `date`
- `task`
- `artifact_type`
- `path`
- `source_script`
- `input_paths`
- `description`
- `notes`

For future work, save simulation caches before plotting from them. Use `results/cache/simulations/` for simulation results, `results/cache/plots/` for plot caches, and `results/tables/` for generated tables.
